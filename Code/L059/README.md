이 과정에서 
1. 프로그램 기본 골격 구성
    t0.py - 데이터 처리 순서별 배치
    t1.py - 반복 테스트를 위한 구조 배치
2. 결과 표시하는 방법
    t2.py - 테스트 결과를 화면에 표시하기
3. 연습 데이터를 문장에서 추출할 수 있도록 하고
    t3.py - 인터넷에서 목록을 복사하여 사용하는 방법
    t4.py - 텍스트 데이터 처리 정규식 소개
4. 대소 문자를 선택적으로 비교할 수 있도록 합니다.
    t5.py - 대소문자 비교 여부 적용 사용자 확인 구조 적용
5. 마지막으로 연습 데이터를 파일 또는 서버로부터 가져오는 방법을 배웁ㄴ디ㅏ.
    t6.py - 파일로부터 데이터를 사용하는 방법
    t7.py - 서버로부터 데이터를 사용하는 방법
기타
    w.py - 서버로부터 데이터를 가져오기 위한 방법 실험
